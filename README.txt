Pacote pronto para Vercel.

Estrutura:
- index.html
- images/
- vercel.json

Como usar no Vercel:
1. Descompacte o ZIP.
2. Suba a pasta inteira no Vercel.
3. O arquivo principal é index.html.

Observação:
Agora o HTML usa imagens externas da pasta images/, o que costuma funcionar melhor no celular do que um HTML único com tudo embutido.
